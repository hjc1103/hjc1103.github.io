神秘海域

发表于 2022-06-30 更新于 2021-08-10

随便写写

Ⅰ

由于那些科学家们在了解事情的原委前不肯听从我的忠告，因此我不得不发表这篇说明。我反对纽约大学即将进行的对于南太平洋海域的大规模科学考察活动，但是我却非常不愿意说出其中的原因。因此，在我将所有事实公布之前，又不会有人相信我的忠告，这又是我更不愿意开口。

当我下定决心公布这些事实时，我的内心刚刚结束一场激烈的心理斗争。不过，如果看到这篇忠告的人能够改变他们疯狂的想法，那么我将本与帕尼尔约定的用不对外说出一个字的秘密公之于世的行为，也终将被神明所宽恕。

可怜的年轻人帕尼尔，我终究也不知道，当我们两人用仅存的理智——抑或是早已失去理智，凭借的只是潜意识中的对于未知的最原始的恐惧，从那海底深渊中逃出时他往后的最后一督中，究竟看见了怎样的事物。自那天以后，我常常见到帕尼尔会坐在椅子上，眼神迷离、精神恍惚，配以口中的喃喃自语。等到他恍惚如从梦中醒来时，又会坚决地对之前说过的话语全盘否认。我曾不止一次地在他陷入“梦中”时，靠近他，并试图听出些什么来。

“犹格·索托斯”“深藏于喜马拉雅之中的米·戈”“永远沉睡的拉莱耶”

以及不断从他嘴中发出的古怪声音：“Tekeli-li! Tekeli-li!”

听到这些晦涩而亵渎神明的古怪名词，我不禁联想到我在密斯卡托尼克大学图书馆那阴暗潮湿的地下室中读过的那本，由阿拉伯疯子阿卜杜·阿尔哈兹莱德所撰的《死灵之书》中也只敢模糊叙述的恐怖存在。在这一刻，我开始后悔自己阅读了那本令人嫌恶的《死灵之书》，后悔与大学里那位博学得甚至有些令人不快的民俗学者威尔马斯过多地讨论这些东西。

如果您对这类科学考察活动比较感兴趣，您一定听说过或在报纸上看到过不久前由我，程岩教授，带领的密斯卡托尼克大学的考察团在南太平洋某海域调查时的“意外收获”。这次考察对外的转播都经过的很大部分的删节。并且最后的我与年轻的帕尼尔两人前往海底遗迹中考察的大部分情节均经过伪造，可以说是没有一个字是真实的。但即使是这样的结果依旧吸引了许多早已对这片海域感兴趣的学者们。为此，我不得不再度回忆起两个月前的如梦境碎片般的回忆——因为当时我与年轻的帕尼尔均仅剩最后一丝理智，并且我眼前所见的生物——并不能准确地定义为生物，具体的原因我会在后面再进行叙述——远远超乎了迄今为止每个人类想象的极限。

Ⅱ

在考察过程中，我们向《阿卡姆广告人》发送了许多简报。探险回来后，我与本次探险的工程师——来自工程系的比利菁也写了不少记录来记录仪本次探险。出于这些，公众已对密斯卡托尼克大学探险队有所了解。我们这支队伍里包括了四位来自密斯卡托尼克大学的专业人士——比利，生物系的荼虞，来自物理系并兼职气象专家的伍德，还有我这个名义上的总指挥。除此之外还有十几位研究生助手。

本次的考察活动由几个基金会提供支持，所以虽然我们的影响并不大但是资金算是很充足。

我们一行于1965年7月15日从密斯卡托尼克大学出发，计划在冬季到来之前返回。我们从洛杉矶港搭乘轮船“阿卡姆”号和“渔人”号向着南太平洋进发。由于我们之中只有船长利普先生有去南太平洋附近航行的经历，所以旅程中他的作用显得举足轻重。

清明节

又是一年清明，最期待的还是比周末多放一天的快乐。这种期待从高二下班学期开学以来就一直存在了。等到真正放假的时候又不是很快乐。人们都是这样还是只有我是这样，谁知道呢？

写这篇日记的时候已经是四月四号的深夜。完全是出于内心的表达欲让我重新点开大半年没有更新的博客。

说是“表达欲”，其实并没有那么的准确。因为我写的东西压根就只有我自己会看。无所谓了，当初创建这里的时候我的心里就想到这一点了。当时满心踌躇创建了一个自己的博客，其实大部分原因是因为好奇心吧。发布第一篇文章的时候内心充斥着自豪感，就连现在也是。这也是“表达欲”得到实现的证明吧？

清明代表着春天的真正到来。春天是万物复苏的季节，当然也是动物们蠢蠢欲动的季节。奇怪的是新的恋爱情况出现最频繁的时候是每年的元旦附近，或许是天太冷了想要抱团取暖？蛤蛤蛤谁知道呢。

下一次我自己来这里并且看到这一篇博客的时候又过去了多久呢？我想问你，最近成绩怎么样？三分球是否还拥有准星？答应自己的假期训练到底练了没？写到这里时内心住着的女孩现在是否依偎着你肩？

如果都是肯定的回答，那该多好啊。反正现在的我可是一项都没达成呢！

经过一天的研究，我又学到了怎样添加更加高级的看板娘！下面我来详细介绍一哈。

终极–互动看板娘

第一步

下载大神的文件并解压到本地博客目录的themes/next/source下，修改autoload.js文件。

将

const live2d_path = "https://cdn.jsdelivr.net/gh/stevenjoezhang/live2d-widget/";

改为

const live2d_path = "/live2d/";

autoload.js中的注释的绝对地址指的是，将资源打包放到hexo/theme/next/source中后，以hexo/theme/next/source为根目录（/）的绝对路径，一般我们下载下来的是master分支，那么修改路径就是/live2d-widget-master/xxx.js，autoload.js的最后一个函数initWidget;中的第二个参数不要变，是一个后台api。我选择将文件夹重命名为“live2d”，因此按照如上方式修改。

第二步

在/themes/next/layout/_layout.swing中,标签中新增如下内容，########为你自己的github账号：

<script src="https://########.github.io/live2d/autoload.js"></script>

标签中新增如下内容：一定一定要加上依赖！！！！！

<script src="https://cdn.jsdelivr.net/npm/jquery/dist/jquery.min.js"></script>
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/font-awesome/css/font-awesome.min.css"/>

第三步

在主题配置文件_config.yml中,新增如下内容：

live2d:
  enable: true

想修改看板娘大小、位置、格式、文本内容等，可查看并修改 waifu-tips.js 、 waifu-tips.json 和 waifu.css。

最终的效果在我的博客就能看到啦！

今天来试验一下看板娘功能！

看板娘

首先，用如下的命令添加一个live2d

输入以下的命令，下载相应的模型

模型名称选择请单击此处各个模型的预览请访问原作者的博客

打开站点目录下的_config.yml文件，输入以下代码

live2d:
enable: true
scriptFrom: local
model: 
    use: live2d-widget-model-haruto #模型选择
display: 
    position: right  #模型位置
    width: 150       #模型宽度
    height: 300      #模型高度
mobile: 
    show: true      #是否在手机端显示

设置好之后我们便拥有了一个卡通人物
